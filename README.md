# Waveform Audio Player
Este projeto é um player de áudio com visualização de waveform, desenvolvido em React, TypeScript e [wavesurfer](https://github.com/katspaugh/wavesurfer.js).

<img width="1023" height="898" alt="image" src="https://github.com/user-attachments/assets/570253b3-ccc8-4155-bc88-428db6e0ba5d" />

## Funcionalidades

- Reprodução de arquivos de áudio
- Visualização gráfica do waveform
- Interface moderna e responsiva
